Step 3: Configure Project Credentials

üîß Configure your Supabase credentials

Get your project URL and API key from Supabase dashboard.

Get your credentials:

  1. In your Supabase project, go to Settings ‚Üí API
  2. Copy your Project URL
  3. Copy your anon public key (not the service_role key!)
  4. Open js/supabase.js in your code editor
  5. Replace YOUR_SUPABASE_PROJECT_URL with your Project URL
  6. Replace YOUR_SUPABASE_ANON_KEY with your anon public key
  7. Save the file
